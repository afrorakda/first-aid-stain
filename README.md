# #63 First Aid Stain

A hyper-minimalist emergency guide for common fabric stains. Designed for the panicked "Maman" (or anyone) who needs immediate, 1-step instructions to save a garment before the stain sets.

## Core Philosophy
* **1 Tool / 1 Action**: No searching, no long articles. Tap the stain type and act instantly.
* **First Aid Focus**: Prioritizes immediate field-action (what to do in the first 60 seconds).
* **Safe-First Logic**: Explicitly highlights "DON'T" actions (like using hot water) to prevent permanent damage.

## Key Features
* **Zero-Search Interface**: A clean grid of 15 common stain types for instant visual recognition.
* **DO/DON'T/ACTION Structure**: Minimalist guidance using only household items (cold water, soap, salt).
* **Safety Fail-safe**: The "General Rule" footer ensures users have a safe default action for unknown stains.
* **Kitchen-Proof UI**: Large, high-contrast buttons and 15px rounded-corner design for easy operation under stress.

## Design System
* **Background**: #fff
* **Main Accent**: #333
* **Status Colors**: Success Green (#27ae60) / Danger Red (#e74c3c)
* **Border Radius**: 15px
* **iOS Optimization**: Prevents text fading and ensures crisp rendering on mobile Safari.

## Privacy & Performance
* **No Tracking / No Ads / No Login**
* **Browser-only**: Zero data leaves the device.
* **Ultra-Lightweight**: Fast loading even on weak mobile connections.
